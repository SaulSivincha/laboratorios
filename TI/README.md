# Distribuidora Andina

Proyecto Flutter basico inspirado en el diseno de Stitch:

- Paleta corporativa: Blue Slate, Sea Green, Silver, White y Night Bordeaux.
- Navegacion inferior con 3 pestanas: Inicio, Ruta y Pedidos.
- Variables globales simples para vendedor, ruta, productos y carrito.
- Flujo funcional basico: check-in, marcar visitas, sumar/restar productos y registrar pedido.

## Ejecutar

Flutter quedo instalado para este usuario en:

```bash
/home/sauls/.local/share/flutter
```

Los comandos `flutter` y `dart` estan enlazados desde `/home/sauls/.local/bin`, que ya esta en el `PATH`.

Para ejecutar la app:

```bash
flutter pub get
flutter run -d linux
```

El scaffold nativo ya incluye Linux y Android. Para Android todavia falta instalar/configurar el Android SDK.
